# Directory Submission Quick-Fills — 60 seconds each

Open each directory link, then copy-paste the labelled values into the matching form fields. Verification clicks land in `info@thedripmap.com` — open that inbox first and keep it open in a tab.

**Recommended order**: Brownbook (fastest, no account) → Hotfrog → n49 → Lacartes → Cylex. Skip BBB until ~Nov 2026 (6-month business-age rule). Skip YellowPages / Yelp / Canada411 / 411.ca for now (they require phone verification — separate session).
